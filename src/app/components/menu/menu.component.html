Aplicación construida con FLASK en el backend y ANGULAR 6+ para el frontend, que permite crear peliculas y almacenarlas en una base de datos SQLITE y además permite agregar las imagenes que desee a cada película; cada imagen se agrega a un bucket S3 de AWS de donde se extraen por URL para su visualización.
